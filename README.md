A full-stack food ordering web application for canteens, built with **Node.js**, **Express**, **EJS**, **PostgreSQL**, and hosted on **Render**. Users can browse the menu, place orders, and admins can manage menu items and orders.

### User Features
- Browse food menu with categories.

### Admin Features
- Admin login and session-based authentication.
- Add, delete, and manage menu items.
- Add and delete categories.
- View all orders with user details and ordered items.
